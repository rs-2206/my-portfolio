# My Portfolio

This is a simple portfolio website created using HTML, CSS, and JavaScript. It showcases my projects, skills, and contact information.

## Project Structure

```
my-portfolio
├── index.html        # Main HTML document for the portfolio
├── css
│   └── styles.css    # Styles for the portfolio website
├── js
│   └── script.js     # JavaScript for interactivity
├── assets
│   └── projects.json  # JSON file containing project details
└── README.md         # Documentation for the project
```

## Features

- Responsive design
- Interactive elements using JavaScript
- Dynamic project display from JSON data

## Setup Instructions

1. Clone the repository or download the files.
2. Open `index.html` in your web browser to view the portfolio.
3. Customize the content in `projects.json` to showcase your own projects.

## Technologies Used

- HTML
- CSS
- JavaScript
